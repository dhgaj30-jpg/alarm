# مشروع مواقيت الصلاة - Kotlin / Android

هذا مشروع أندرويد عربي بسيط لكنه كامل كبداية قوية، ويحتوي على:

- عرض مواقيت الفجر والظهر والعصر والمغرب والعشاء
- اختيار الدولة: سوريا، الأردن، أو تلقائي عبر GPS
- اختيار الصلاة التي تريد تنبيهها
- تقديم أو تأخير من 1 إلى 60 دقيقة
- نغمة أذان مخصصة من التخزين عبر منتقي الملفات
- إشعار + تشغيل صوت عند الوقت المحدد
- غفوة 2 أو 5 دقائق
- إعادة تنبيه كل 2 أو 5 دقائق بعد الإيقاف
- إعادة جدولة بعد إعادة تشغيل الهاتف

## الملفات المهمة
- `MainActivity.kt` : شاشة التطبيق
- `MainViewModel.kt` : إدارة الحالة
- `SettingsRepository.kt` : حفظ الإعدادات في DataStore
- `PrayerTimeCalculator.kt` : حساب المواقيت
- `LocationProvider.kt` : جلب الموقع الحالي
- `AlarmScheduler.kt` : جدولة المنبه الدقيق
- `PrayerAlarmReceiver.kt` : تشغيل المنبه عند الوقت
- `AlarmSoundService.kt` : تشغيل صوت الأذان بالخلفية
- `AlarmActionReceiver.kt` : الغفوة والإيقاف
- `BootReceiver.kt` : إعادة الجدولة بعد التشغيل

## الملاحظات
1. هذا المشروع مضبوط على `minSdk 26` لتسهيل التعامل مع الوقت والتواريخ.
2. على Android 13+ يجب طلب إذن الإشعارات.
3. على Android 12+ قد تحتاج إلى فتح إذن المنبهات الدقيقة من الإعدادات.
4. اختيار النغمة يتم عبر `ACTION_OPEN_DOCUMENT` مع حفظ الصلاحية.

## كيف تبنيه
- افتح المشروع في Android Studio
- انتظر Gradle Sync
- شغّل التطبيق على جهاز حقيقي
- امنح أذونات الموقع والإشعارات
- اختر النغمة من التخزين
- اضغط حفظ وتفعيل المنبه

## ملاحظات تطويرية
- يمكن لاحقًا إضافة:
  - Widget للشاشة الرئيسية
